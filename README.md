Project Overview: Lumière Premium Nail Salon

This project is a high-fidelity, single-page landing page designed for Lumière, a luxury nail studio. It was developed with a focus on a "quiet luxury" aesthetic, utilizing a minimalist layout and a sophisticated palette of bone, sand, and deep mauve tones.

Key Technical & Design Features
Premium Visual Identity: The site uses a blend of Cormorant Garamond for an elegant serif feel and DM Sans for modern, readable UI elements.

Immersive Hero Experience: The entry point features a full-screen background video with a linear gradient overlay to ensure text legibility while maintaining a high-end atmosphere.

Custom Design System: Built using CSS custom properties (variables) for a centralized color and spacing system, including specific "card tokens" inspired by modern UI libraries like shadcn/ui.

Performance-Driven Animations: A custom-built Scroll Reveal system leverages the Intersection Observer API to trigger smooth entrance animations as users navigate the page.

Mobile-First Functionality: The site includes a fully responsive navigation bar that transforms into a blur-effect mobile menu with a custom-animated burger icon.

Interactive Components:

Service Menu: A clean, hover-responsive list of studio offerings and pricing.

Social Proof: A multi-layout testimonial grid featuring diverse card styles.

Booking Engine: A front-end appointment request form with built-in validation feedback and success states.

Tech Stack
Structure: Semantic HTML5.

Styling: Modern CSS3 with Flexbox, CSS Grid, and custom variables.

Logic: Vanilla JavaScript for scroll detection, menu toggles, and animation triggers.

This project serves as a showcase of modern front-end development, balancing aesthetic elegance with functional performance.
